Review notes

Traffic Guard is a menu bar utility for users who connect their Mac to a limited mobile hotspot plan. It reads local macOS network diagnostics to estimate app traffic and show alerts. Optional Chrome tab access helps identify the browser tab behind Chrome traffic.

Privacy label draft

Data used locally

Usage counters, Wi-Fi names, app names, process names, endpoint hostnames or IP addresses, ports, alert history, and app protection settings.

Optional browser details

Chrome tab titles and URLs may be read with user permission to help identify active browser traffic.

Product analytics

Configured release builds may use Firebase Analytics for app launches, screen views, settings changes, alerts, feedback submit results, and app protection actions. The app does not include advertising SDKs.

Optional feedback

When users choose Send Feedback, their message, optional reply email, app version, and macOS version are sent to Traffic Guard Support. Wi-Fi, quota, traffic engine, and top app usage details are sent only when the user selects Include usage diagnostics.

User control

Users can change thresholds, hotspot names, app rules, and reset local usage baselines.

Permission wording

Notifications: Traffic Guard sends alerts when Wi-Fi hotspot data usage approaches the configured quota.

Apple Events: Traffic Guard reads Google Chrome tab titles and URLs to help identify which tab may be using hotspot data.
